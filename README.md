# telegram-bot

Телеграм бот, который реализует общение с человеком посредством API телеграм бота.

Телеграм бот умеет:

- отвечать на сообщения
- присылать информацию
- закреплять сообщения
- работать с файлами.

Телеграм бот можно и нужно развивать, есть возможность интегрировать:

- работу с сервером
- запись данных пользователя в БД
- расширение функционала.

Телеграм бот можно внедрить в свою платформу для быстрой и удобной работы с пользователями, их обращениями и данными.

## Запуск

Корневой каталог: /app

Для работы с телеграм ботом требуется:

1. Данные из папки /app поместить в корень
2. В файл /app/token.py в поле TOKEN ввести токен\*

Токен\* телеграм бота — уникальная строка, благодаря которой есть возможность обращаться к конкретному боту, получается
с помощью бота телеграм BotFather.
